# PROJECT

## 프로젝트명
한국 삼국 전략 RPG (가칭)

## 장르
수집형 영웅 육성 + 전략 전투 + 성 운영

## 플랫폼
모바일 우선 (iOS/Android), 이후 확장 가능 구조

## 타겟
- 한국 삼국/역사 소재를 좋아하는 플레이어
- 영웅 수집 게임에서 박탈감을 느껴본 플레이어
- 메타 강요보다 전략적 선택과 운영 재미를 선호하는 플레이어

## 핵심 컨셉
- 모든 영웅은 사용 이유가 존재한다.
- 영웅의 가치는 전투력 숫자만으로 결정되지 않는다.
- 콘텐츠가 영웅의 가치를 만들고, 플레이어가 조합의 의미를 찾는다.

## 프로젝트 목표
- SSR 중심 고착 메타에서 벗어나 다양한 영웅 활용 구조를 만든다.
- 운빨/독식/창고 캐릭터로 인한 박탈감을 줄이고 지속 플레이 동기를 만든다.
- "좋아하는 장수를 끝까지 쓰고 싶다"는 욕구를 시스템적으로 보장한다.
- 전투 외 성 운영/콘텐츠 참여를 통해 영웅의 역할을 확장한다.
