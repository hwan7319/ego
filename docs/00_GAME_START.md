# GAME START

## 목적
게임 시작 10분 안에 플레이어가 "나는 EGO다"를 명확히 체감하게 만든다.

## 시작 선언
- EGO는 왕이 아니다.
- EGO는 신이 아니다.
- EGO는 시간여행자가 아니다.
- EGO는 세계 안에서 살아가는 플레이어 자신이다.

## 초기 경험 목표
1. 나는 누구인가? -> EGO(플레이어 자신)
2. 왜 싸우는가? -> 세계 안에서 살아남고 역사를 바꾸기 위해
3. 어떻게 성장하는가? -> 내가 선택한 직업/능력/역할로

## Game Start Flow
### Step 1. EGO 생성
- 플레이어는 이름, 기본 성향, 시작 전투 성향(예: 공격/균형/지원)을 설정한다.
- EGO는 가챠 대상이 아니며 계정의 중심 캐릭터로 고정된다.

### Step 2. 첫 전투 참여
- EGO는 지휘관 UI만 사용하는 존재가 아니라, 6인 진형 중 1인으로 직접 전투에 참여한다.
- 이 단계에서 플레이어는 "EGO도 전투 유닛"이라는 규칙을 학습한다.

### Step 3. 첫 영웅 합류
- 광개토대왕, 김유신, 계백, 을지문덕 등 역사 영웅이 동료로 합류한다.
- 영웅은 플레이어의 분신이 아니라, 플레이어와 함께 싸우는 동료로 소개한다.

### Step 4. 첫 성장 선택
- EGO의 직업/능력/역할 중 첫 분기 선택을 제공한다.
- 같은 EGO라도 플레이어 선택에 따라 완전히 다른 전투 정체성이 형성되기 시작한다.

## 설계 원칙
- 게임의 중심은 "영웅 선택"이 아니라 "플레이어 정체성 확립"이다.
- EGO는 Hero System이나 World의 하위 항목이 아니라, 전 시스템을 관통하는 중심축이다.
