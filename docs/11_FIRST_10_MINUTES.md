# EGO First 10 Minutes

## 목표
첫 10분이 10년의 기억이 되도록 한다.

## 기존 게임의 첫 경험
설명
-> SSR 지급
-> 뽑기
-> 전투
-> 강화

플레이어는 시스템을 경험한다.

## EGO의 첫 경험
낯선 시대
-> 한 사람과의 만남
-> 작은 사건 경험
-> 첫 번째 관계 형성
-> 앞으로 함께할 역사 시작

플레이어는 이야기를 경험한다.

## 첫 10분 원칙
### 하지 않는다
- 첫 화면부터 가챠 강조
- 전투력 비교
- 과금 유도
- 반복적인 튜토리얼

### 보여준다
- 한 명의 사람
- 하나의 사건
- 하나의 선택
- 하나의 기억

## EGO의 시작 질문
게임은 플레이어에게 묻는다.

- "당신은 선택받은 영웅인가?"가 아니다.
- "당신은 어떤 사람이 될 것인가?"를 묻는다.

## 최종 방향성
EGO는 위대한 영웅이 되는 게임이 아니다.
위대한 영웅들과 함께 살아가며, 나만의 역사를 만들어 가는 게임이다.
