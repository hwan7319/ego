# EGO Growth Philosophy

## 1) EGO의 성장은 무엇인가
기존 RPG에서 성장의 목적:

레벨 상승  
-> 능력치 증가  
-> 강해짐  
-> 최종 도달

하지만 EGO의 성장은 다르다.

EGO는 완성되는 존재가 아니다.  
플레이어의 선택과 경험에 따라 계속 변화하는 존재이다.

### 핵심 문장
EGO는 성장하는 것이 아니라 계속 만들어진다.

## 2) EGO에는 완성이 존재하지 않는다
플레이어의 자아에는 정답이 없다.  
최고의 EGO라는 기준도 존재하지 않는다.  
강함만으로 평가할 수 없다.

일반적인 게임:

최종 목표 달성  
-> 완성

EGO:

경험  
-> 변화  
-> 새로운 선택  
-> 새로운 나

### EGO의 목표
- 최고가 되는 것: 아니다.
- 완성되는 것: 아니다.
- 나만의 역사를 만들어 가는 것: 맞다.

## 3) 플레이어는 왜 EGO의 여정을 계속하는가
질문: 왜 사람은 계속 살아가는가?

답:
- 아직 경험하지 못한 것이 있기 때문에
- 아직 만나지 못한 사람이 있기 때문에
- 아직 가보지 못한 곳이 있기 때문에
- 아직 발견하지 못한 내가 있기 때문에

EGO에서 이 이유들은 콘텐츠가 된다.

### EGO Journey Motivation
궁금함  
-> 만남  
-> 경험  
-> 선택  
-> 변화  
-> 새로운 궁금함

## 4) EGO 장기 서비스 방향
기존 모바일 RPG:

신규 캐릭터 출시  
-> 기존 캐릭터 가치 하락  
-> 새로운 캐릭터 필요

EGO:

새로운 인물 등장  
-> 새로운 가치관 등장  
-> 새로운 성장 방향 발견  
-> 기존 영웅 재해석

### 신규 영웅의 의미
- 새로운 영웅 = 더 강한 캐릭터: 아니다.
- 새로운 영웅 = 새로운 선택지: 맞다.

## 5) EGO 성장 구조
EGO는 세 가지 요소가 연결된다.

영웅과의 관계  
+ 역사적 사건 경험  
+ 플레이어의 선택  
= 나만의 EGO

## 6) 영웅과의 관계
영웅은 단순한 수집 대상이 아니다.

기존 수집형 RPG:

획득  
-> 강화  
-> 사용

EGO:

만남  
-> 이해  
-> 신뢰  
-> 배움  
-> 변화

플레이어는 영웅을 수집하는 것이 아니다.  
사람을 만난다.

## 7) 깨달음 시스템 (가칭)
EGO의 성장은 숫자가 아니다.  
경험을 통해 새로운 가능성을 얻는다.

예시: 계백과의 경험

상황: "성을 지킬 것인가, 후퇴할 것인가?"

선택 A: 끝까지 지킨다.
- 획득: 깨달음 `희생`
- 효과: 방어 강화, 보호 능력 증가, 위기 대응 강화

선택 B: 살아서 다시 싸운다.
- 획득: 깨달음 `재기의 의지`
- 효과: 회복, 재정비, 전략 강화

역사적 정답은 존재하지 않는다.  
역사는 배경이다.  
플레이어의 선택이 새로운 역사가 된다.

## 최종 정의
EGO는 위대한 영웅이 되는 게임이 아니다.  
위대한 영웅들과 만나며, 나만의 역사를 만들어 가는 게임이다.

## 오늘 회의 핵심 결정사항 (EGO Growth Rule)
| 항목 | 방향 |
| --- | --- |
| 성장의 목표 | 완성이 아닌 변화 |
| 성장의 기준 | 전투력이 아닌 경험 |
| 성장의 재료 | 만남, 선택, 기억 |
| 최종 형태 | 플레이어마다 다른 EGO |
| 서비스 방향 | 새로운 가능성을 지속 제공 |
