# ROOT

> 플레이어는 여러 영웅의 가치관과 전투 철학을 배우며 자신만의 자아(EGO)를 완성해 간다.
> 영웅을 수집하는 게임이 아니다. 영웅에게 배우는 게임이다.

## EGO Philosophy
### 1) EGO는 무엇인가
EGO는 특정 영웅이나 왕의 이름이 아니다.

EGO는 플레이어 자신이며, 수많은 역사적 인물들과 만나고 경험하며 만들어 가는 나만의 자아(Self)다.

### 2) EGO가 추구하는 경험
기존 영웅 서사:

평범한 사람
-> 선택받은 존재
-> 특별한 힘 획득
-> 영웅이 됨

EGO의 방향:

나 자신
-> 역사 속 인물들과 만남
-> 그들의 가치관과 철학을 경험
-> 수많은 선택을 통해 성장
-> 나만의 역사 완성

### 핵심 문장
EGO는 특별한 존재가 되는 이야기가 아니다.
나 자신으로서 역사를 살아가는 이야기다.

### EGO의 플레이어 정의
- 플레이어는 선택받은 영웅이 아니다.
- 플레이어는 역사를 살아가는 한 사람이다.
- 플레이어는 영웅들에게서 배운다.
- 하지만 플레이어는 그들이 되지 않는다.
- 영웅들의 경험은 플레이어의 일부가 되고, 그 선택들이 모여 나만의 EGO가 완성된다.

예시 학습 축:
- 광개토대왕에게 개척의 의지를 배운다.
- 을지문덕에게 지혜와 전략을 배운다.
- 계백에게 신념과 책임을 배운다.

### EGO 관계 철학
기존 수집형 RPG:

영웅 획득
-> 강화
-> 전투력 증가

EGO:

만남
-> 관찰
-> 이해
-> 신뢰
-> 동료가 됨

플레이어가 수집하는 것은 영웅이 아니다.
함께 만들어낸 기억이다.

### 영웅과 플레이어의 관계
영웅은 플레이어를 하나의 관계로 정의한다.

- 스승: 플레이어에게 새로운 방향을 보여주는 존재
- 동료: 같은 전장에서 함께 싸우는 존재
- 제자: 플레이어에게 새로운 시대를 배우는 존재

최종 관계 정의:
영웅과 플레이어는 서로에게 배우는 관계이다.

### 장르 정체성 전환
초기 EGO:
"삼국지 영웅을 수집하는 게임"

현재 EGO:
"삼국지라는 시대 속에서 나라는 사람이 성장하는 게임"

## 왜 이 게임을 만드는가
기존 수집형 게임의 반복된 박탈감 구조(SSR 고착, 운빨, 길드 독식, 창고 캐릭터)를 줄이고,
플레이어가 애정 캐릭터를 포기하지 않아도 되는 경험을 만들기 위해서다.

## 왜 한국 삼국인가
한국 삼국은 익숙하면서도 게임적으로 재해석할 여지가 큰 소재다.
지역/세력/인물의 관계를 활용해 전투와 운영 양쪽에 의미 있는 선택을 설계할 수 있다.

## 왜 이 프로젝트인가
이 프로젝트는 단순한 영웅 수집 게임이 아니라,
"영웅 가치의 공정성"을 중심에 둔 구조 실험이라는 점에서 팀의 방향성과 맞다.

## 플레이어는 누구인가
- 역사 기반 세계관을 좋아하는 플레이어
- 특정 메타를 강요받는 경험에 피로를 느낀 플레이어
- 조합, 운영, 장기 성장을 함께 즐기고 싶은 플레이어

## 이 게임이 존재해야 하는 이유
강한 것만 남는 구조가 아니라,
필요한 것이 살아남는 구조를 통해 플레이어의 선택과 애정을 보존하는 게임이 필요하다.
